# 🗺️ MacroSignal — PLAN (마스터 실행 계획)

| 항목 | 내용 |
|---|---|
| 문서 버전 | v1.0 |
| 역할 | 전체 문서를 묶는 최상위 실행 계획. 마일스톤·의존성·리스크·의사결정 총괄 |

---

## 1. 문서 인덱스 (전체 산출물 맵)

| 문서 | 역할 | 상태 |
|---|---|---|
| 사전계획서 | 데이터 소스·로직 조사 | ✅ |
| **PRD** | 무엇을/왜 (2트랙·3시장·실시간정책) | ✅ |
| **TRD** | 어떻게 (아키텍처·API명세·폴백) | ✅ |
| **ERD** | 데이터 (스키마·RLS·시드) | ✅ |
| **GUIDE** | 세팅·실행·배포·확장 | ✅ |
| **TASK** | 한 번에 하나씩 실행 목록 | ✅ |
| **PROMPT** | AI 브리핑 + 빌드 프롬프트 | ✅ |
| **PLAN** | 마스터 일정·총괄 (본 문서) | ✅ |

> 개발 착수 순서: PLAN 확인 → TASK의 T0-1부터 → 각 태스크마다 PROMPT의 해당 빌드 프롬프트 사용.

---

## 2. 제품 요약 (한 페이지)

- **무엇**: 한·미 금리 / 한·미·일 환율 / 국제 유가를 종합해 **코스피·코스닥·나스닥 3개 시장**의 오늘 장 흐름을 매일 아침 진단하는 어드바이저.
- **차별점**: 같은 거시 뉴스에도 세 시장이 다르게 반응 → **시장별 분화 진단**("코스피🟡·코스닥🔴·나스닥🟢").
- **핵심 통찰**: 일간 신호는 정책금리가 아니라 **국채금리·환율·유가·미선물**. "내일 장 = 밤사이 미국".
- **전략**: 2트랙 (개인용 먼저 → 공개용 확장), 같은 코드베이스로 호환 유지.

## 4. 의존성 그래프 (임계 경로)

```
M0 ──► M1 ──► M2 ──► M3 ──► M4 ──► M5 ══(검증 통과)══► M6
                │              │
          (엔진은 M1의        (UI는 M2·M3
           데이터 필요)        결과 필요)
```
- **임계 경로**: 데이터(M1) → 엔진(M2)이 병목. UI는 그 뒤.
- M6는 M5 실사용 검증 **통과가 게이트**(성급한 공개 금지).

## 5. 트랙 전환 게이트 (M5 → M6 체크리스트)

## 7. 열린 의사결정 (착수 전 확정 권장)

1. **국내 지수 소스**: KIS vs 키움 → 기존 자산·안정성 기준 택일
2. **상관 재계산 주기**: 주간(권장) vs 월간
3. **market_weights 초기값**: 문헌기반 시작 → 백테스트로 조정
4. **임계값 T**: 초기 보수적 설정 후 실사용 캘리브레이션
5. **트랙2 수익모델**: 구독 vs 프리미엄 vs 광고 (M6에서 확정)

## 8. 성공의 정의
